# Extract unique labels from an atlas

Extract unique labels from an atlas

## Usage

``` r
atlas_labels(x)

# S3 method for class 'ggseg_atlas'
atlas_labels(x)

# S3 method for class 'brain_atlas'
atlas_labels(x)

brain_labels(x)
```

## Arguments

- x:

  brain atlas

## Value

Character vector of atlas region labels
