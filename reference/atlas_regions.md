# Extract unique region names from an atlas

Extract unique region names from an atlas

## Usage

``` r
atlas_regions(x)

# S3 method for class 'ggseg_atlas'
atlas_regions(x)

# S3 method for class 'brain_atlas'
atlas_regions(x)

# S3 method for class 'data.frame'
atlas_regions(x)

brain_regions(x)
```

## Arguments

- x:

  brain atlas

## Value

Character vector of region names
