# Extract unique names of brain regions

Convenience function to extract names of brain regions from a
[`brain_atlas`](brain_atlas.md)

## Usage

``` r
brain_regions(x)

# S3 method for class 'ggseg_atlas'
brain_regions(x)

# S3 method for class 'brain_atlas'
brain_regions(x)

# S3 method for class 'data.frame'
brain_regions(x)
```

## Arguments

- x:

  brain atlas

## Value

Character vector of brain region names
